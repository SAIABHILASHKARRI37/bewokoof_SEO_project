## Project Overview
This project is a demo SEO analytics dashboard created for Bewakoof using Looker Studio and Google Analytics-style ecommerce data.

The dashboard focuses on:
- Website traffic trend analysis
- Channel-wise traffic split
- SEO keyword performance
- CTR analysis
- Purchase and revenue tracking

## Tools Used
- Google Sheets
- Looker Studio
- SEO Metrics (CTR, Impressions, Clicks)
- Google Analytics style KPIs

## Key Insights
- Organic Search is the major traffic source
- CTR is 11.02%, indicating strong keyword relevance
- Traffic recovered significantly after Apr 6
- SEO contributes directly to revenue and conversions

## Business Impact
This dashboard helps identify:
- top-performing channels
- keyword opportunities
- traffic drops
- revenue contribution from organic traffic
